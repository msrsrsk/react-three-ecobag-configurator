## <a name="about">🤖 サイトの概要</a>

エコバッグの色やプリント画像を自由にカスタマイズできる 3D コンフィグレーターです。<br />
エコバッグのモデルは Blender で作成し、Three.js でオプションの
カスタマイズができるようになっています。<br />
また、アップロードされた画像の上下左右の位置や大きさなども変更ができるように制作しました。<br />
<br />
制作時期：2024年5月頃

## <a name="more">📙 あとがき</a>

実装できたことがとても面白かったです。<br />
